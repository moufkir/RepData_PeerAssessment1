Reproducible Research: Peer Assessment 1

Introduction

The goal of this assignment is to generate a reproducible document that prepare, analyze and explain data about personal movement using activity monitoring devices such as a Fitbit, Nike Fuelband, or Jawbone Up.

This type of devices collects anonymous data at 5 minute intervals through out the day during two months (October and November, 2012 and include the number of steps taken in 5 minute intervals each day).

  • Dataset: Activity monitoring data [52K]

The variables included in this dataset are:

  • steps: Number of steps taking in a 5-minute interval (missing values are coded as NA)

  • date: The date on which the measurement was taken in YYYY-MM-DD format

  • interval: Identifier for the 5-minute interval in which measurement was taken
